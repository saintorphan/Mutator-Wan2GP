# Mutator leans on what Wan2GP already ships; its only genuinely additive dep is
# a fallback way to locate an ffmpeg binary:
#   - ffmpeg / ffprobe (system binaries; the host sets FFMPEG_BINARY/FFPROBE_BINARY).
#     imageio-ffmpeg is a last-resort fallback so a standalone install can still
#     locate an ffmpeg binary if none is on PATH.
#   - Pillow is used for filmstrip / thumbnail compositing but is HOST-PROVIDED, so
#     it is intentionally NOT re-declared here (re-pinning host deps can disturb
#     Wan2GP's resolved versions).
#   - gradio / gradio_rangeslider (the dual-handle trim slider) are host-provided
#     and optional — Mutator falls back to two plain sliders without the rangeslider.
imageio-ffmpeg
